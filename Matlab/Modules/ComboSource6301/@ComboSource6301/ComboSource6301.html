ComboSource6301

Control interface for Arroyo Instruments ComboSource 6301 laser controller

Description

ComboSource6301 provides a MATLAB interface to control the Arroyo Instruments ComboSource 6301 laser controller via RS-232 using VisaIF. The class handles all Arroyo custom commands for laser output control, current configuration, TEC temperature control, and safety features.

CRITICAL: This device uses Arroyo custom commands, NOT SCPI!
Serial Configuration: COM1 (ASRL1), 9600 baud, 8N1, CR terminator only (NOT CR/LF)
Laser Safety Warning: This device controls a Class 3B/4 laser. Always follow proper laser safety procedures, wear appropriate protective equipment, and ensure the enclosure/interlock is properly closed before enabling laser output.

Syntax

myLaser = ComboSource6301('combosource')
myLaser = ComboSource6301('combosource', 'Other')
myLaser = ComboSource6301('combosource', 'Other', false)
Important: The interface parameter must be 'Other' for Arroyo devices. The default 'RS232' interface type does not work with Arroyo custom commands.

Device Information

Method Description
getID() Get device identification string (Arroyo command: *IDN?)

getSerialNumber() Get device serial number (Arroyo command: SN?)

getErrorString() Get error message string (Arroyo command: ERRSTR?)

Laser Current Control

Method Description
setLaserCurrent(current_mA) Set laser drive current in milliamperes (Arroyo: LAS:LDI <value>)
getLaserCurrent() Query laser drive current in mA (Arroyo: LAS:LDI?)
setLaserCurrentLimit(limit_mA) Set laser current compliance limit in mA (Arroyo: LAS:LIM:LDI <value>)
getLaserCurrentLimit() Query laser current limit in mA (Arroyo: LAS:LIM:LDI?)

disableLaser() Disable laser output for safety (Arroyo: LAS:OUT 0)

Examples

Basic Connection and Identification

% Connect to laser controller (must use 'Other' interface)
myLaser = ComboSource6301('combosource', 'Other');

% Get device information using Arroyo commands
idString = myLaser.getID();
version = myLaser.getVersion();
serialNum = myLaser.getSerialNumber();

fprintf('Device: %s\n', idString);
fprintf('Firmware: %s\n', version);
fprintf('Serial: %s\n', serialNum);

% Check for errors
errCode = myLaser.getError();
if errCode ~= 0
    errStr = myLaser.getErrorString();
    warning('Device error %d: %s', errCode, errStr);
end

% Close connection
myLaser.delete();

Basic Laser Current Control

myLaser = ComboSource6301('combosource', 'Other');

% Set laser current limit (safety)
myLaser.setLaserCurrentLimit(50);  % 50 mA maximum

% Set laser drive current
myLaser.setLaserCurrent(25);       % 25 mA setpoint

% Verify settings
limit = myLaser.getLaserCurrentLimit();
current = myLaser.getLaserCurrent();
fprintf('Limit: %.2f mA, Setpoint: %.2f mA\n', limit, current);

% IMPORTANT: Check enclosure is closed before enabling!
input('Press Enter when enclosure is closed...', 's');

% Enable laser
myLaser.enableLaser();
pause(1);

% Check status
if myLaser.isLaserEnabled()
    fprintf('Laser is ON\n');
else
    fprintf('Laser failed to enable - check enclosure!\n');
end

% Disable laser
myLaser.disableLaser();
myLaser.delete();

TEC Temperature Control (Using Direct Serial)

% NOTE: TEC methods not yet in class - use direct serial commands
myLaser = ComboSource6301('combosource', 'Other');

% Set TEC temperature limits
myLaser.writeString('TEC:LIM:THI 30');  % High limit 30°C
myLaser.writeString('TEC:LIM:TLO 10');  % Low limit 10°C

% Set target temperature
myLaser.writeString('TEC:T 20');  % 20°C target

% Enable TEC
myLaser.writeString('TEC:OUT 1');
pause(0.5);

% Check for errors after enabling
errCode = myLaser.getError();
if errCode ~= 0
    errStr = myLaser.getErrorString();
    fprintf('TEC Error %d: %s\n', errCode, errStr);
    myLaser.delete();
    return;
end

% Monitor temperature for 30 seconds
fprintf('Waiting for temperature stabilization (30s)...\n');
for i = 1:30
    temp = str2double(myLaser.queryString('TEC:T?'));
    setpoint = str2double(myLaser.queryString('TEC:SET:T?'));
    fprintf('Time: %2ds, Temp: %.2f°C, Setpoint: %.2f°C\n', i, temp, setpoint);
    pause(1);
end

% Disable TEC
myLaser.writeString('TEC:OUT 0');
myLaser.delete();

Complete Startup with TEC and Gradual Current Ramp

% See: Howtos/Sonstiges/ComboSource/Howto_laser_startup_with_TEC_and_safety.m
% This complete example includes:
% - TEC temperature control with error checking
% - 30-second stabilization wait
% - User confirmation for enclosure
% - Gradual current ramping (0 to 27.5 mA)
% - Comprehensive error handling (E-202, E-508)
% - Safe shutdown sequence

myLaser = ComboSource6301('combosource', 'Other');

% 1. Configure TEC
myLaser.writeString('TEC:LIM:THI 30');
myLaser.writeString('TEC:LIM:TLO 10');
myLaser.writeString('TEC:T 20');  % 20°C target

% 2. Enable TEC and check for errors
myLaser.writeString('TEC:OUT 1');
pause(0.5);
if myLaser.getError() ~= 0
    error('TEC enable failed: %s', myLaser.getErrorString());
end

% 3. Wait for stabilization (30 seconds)
fprintf('Stabilizing temperature...\n');
pause(30);

% 4. User safety check
response = input('Is enclosure closed? (y/n): ', 's');
if ~strcmpi(response, 'y')
    myLaser.writeString('TEC:OUT 0');
    myLaser.delete();
    return;
end

% 5. Configure laser
myLaser.setLaserCurrentLimit(50);
myLaser.setLaserCurrent(0);

% 6. Enable laser
myLaser.enableLaser();
pause(0.5);
if myLaser.getError() ~= 0
    error('Laser enable failed: %s', myLaser.getErrorString());
end

% 7. Gradual current ramp
fprintf('Ramping current...\n');
for current = 0:0.5:27.5
    myLaser.setLaserCurrent(current);
    fprintf('Current: %.1f mA\n', current);
    pause(0.5);
end

% 8. Safe shutdown
myLaser.disableLaser();
myLaser.writeString('TEC:OUT 0');
myLaser.delete();
Common Errors:
E-202: Laser output cannot be enabled → Check enclosure is closed
E-508: Parameter out of range → Value exceeds hardware limits
E-123: Invalid command → Wrong syntax (not SCPI, use Arroyo commands)

Important Notes

Arroyo Custom Commands (NOT SCPI)

CRITICAL: This device does NOT use standard SCPI commands. It uses Arroyo Instruments custom command syntax. Using SCPI commands will result in E-123 errors.

  • Working: LAS:OUT, LAS:LDI, TEC:T, TEC:OUT, *IDN?, VER?, SN?, ERR?, ERRSTR?
  • NOT Working: OUTP ON, SOUR:CURR, MEAS:CURR? (standard SCPI)

Serial Configuration

  • Port: COM1 (ASRL1::INSTR)
  • Baud Rate: 9600
  • Data Bits: 8
  • Parity: None
  • Stop Bits: 1
  • Terminator: CR only (NOT CR/LF)
  • Interface Type: 'Other' (NOT 'RS232')

Safety Features

  • Enclosure: Must be closed before laser can be enabled (E-202 error otherwise)
  • Temperature Control: TEC must be enabled and stabilized before laser operation
  • Current Limits: Always set laser current limit before enabling output
  • Error Checking: Check for errors after every configuration command
  • Shutdown Sequence: Disable laser first, then TEC, then disconnect

Error Handling

Use getError() and getErrorString() after important operations:

  • E-202: Laser output cannot be enabled → Enclosure not closed
  • E-508: Parameter out of range → Value exceeds hardware limits
  • E-123: Path not found → Invalid command syntax (check for SCPI usage)

Unit Conversions

All methods use convenient units (mA, °C). The Arroyo commands typically expect values directly in these units, but some limits may use different scales. The class handles conversions automatically where needed.

Hardware Requirements

  • Arroyo Instruments ComboSource 6301 laser controller
  • RS-232 serial connection (COM1)
  • Serial cable (DB-9 or USB-to-Serial adapter)
  • MATLAB R2019b or later
  • Instrument Control Toolbox (for serialport object)

Software Requirements

  • VisaIF class (version 3.0.0 or newer)
  • Device must be configured in VisaIF_HTW_Labs.csv
  • Configuration: Arroyo-6301 ;; ArroyoInstruments;; ComboSource6301;; Other ;; visa-serial;; ASRL1::9600.8.1.none.none.CR.CR;; 1e4 ;; 1e4
  • NOTE: NI-VISA is NOT required (uses MATLAB serialport)

Inherited Methods from VisaIF

In addition to the methods above, all VisaIF methods are available for direct serial communication:

  • writeString(command) - Send Arroyo command (e.g., 'TEC:OUT 1')
  • queryString(command) - Send command and read response (e.g., 'TEC:T?')
  • readString() - Read response from device
  • getVisaDeviceInfoString() - Get serial port configuration info
Direct Serial Usage: For commands not yet implemented in the class (e.g., TEC methods), use writeString() and queryString() with Arroyo commands. See Howto files for complete examples.
