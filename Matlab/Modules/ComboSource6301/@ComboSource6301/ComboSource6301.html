ComboSource6301

Arroyo Instruments ComboSource 6301 Laser Controller Driver

Version 2.0.1
Implements the full Arroyo command set with all methods validated against the manual

Description

ComboSource6301 provides complete MATLAB control of the **Arroyo Instruments ComboSource 6301** laser controller via RS-232/USB interface using the VisaIF framework. This driver implements the full Arroyo custom command set according to the Arroyo Computer Interfacing Manual.

LASER SAFETY WARNING:
Class 3B/4 Laser Device - Avoid direct or scattered beam exposure
Always wear appropriate laser safety goggles (wavelength-specific)
Ensure enclosure/interlock is closed before enabling laser

CRITICAL - Arroyo Custom Commands:
This device uses ARROYO custom commands, NOT standard SCPI!
Commands like OUTP, SOUR:CURR, MEAS:POW? DO NOT WORK (return E-123)
Serial: COM1, 9600 baud, CR terminator only (NOT CR/LF!)

Syntax

myLaser = ComboSource6301('Arroyo-6301')
myLaser = ComboSource6301('Arroyo-6301', 'visa-serial')
myLaser = ComboSource6301('Arroyo-6301', 'visa-serial', 'few')

Quick Start Example

% Connect
myLaser = ComboSource6301('Arroyo-6301');

% Get device info
fprintf('Device: %s\n', myLaser.getID());
fprintf('Version: %s\n', myLaser.getVersion());

% Configure and enable TEC
myLaser.setTECModeTemperature();
myLaser.setTemperature(25);  % 25°C
myLaser.enableTEC();
pause(30);  % Wait for stabilization

% Configure laser
myLaser.setLaserCurrentLimit(150);  % 150 mA max
myLaser.setLaserCurrent(100);       % 100 mA

% Check interlock before enabling
if ~myLaser.isInterlockClosed()
    error('Interlock is OPEN! Close enclosure.');
end

% Enable laser
myLaser.enableLaser();

% Monitor
fprintf('Temp: %.2f °C\n', myLaser.getTemperature());
fprintf('TEC: %.3f A\n', myLaser.getTECCurrent());
fprintf('Current: %.2f mA\n', myLaser.getLaserCurrent());

% Safe shutdown
myLaser.disableLaser();
myLaser.disableTEC();
myLaser.delete;

Method Arroyo Command Description

isOverTemp() TEC:COND? Check over-temperature (bits 3 & 12)

TEC:COND Register Bit Definitions

Bit Value Meaning
0 1 Current limit
1 2 Voltage limit
2 4 Sensor limit
3 8 Temperature high limit
4 16 Temperature low limit
5 32 Sensor shorted
6 64 Sensor open
7 128 TEC open circuit
12 4096 Thermal run-away
isOverTemp() checks both bit 3 (temp high) and bit 12 (thermal run-away) for complete safety coverage.

Common Errors

Error Code Description Solution
E-202 Laser output cannot be enabled Close enclosure/interlock completely
E-508 Parameter out of range Value exceeds hardware limits - check limits first
E-123 Path not found / Invalid command Using SCPI instead of Arroyo commands - use correct syntax

Configuration Requirements

  • Port: COM1 (ASRL1::INSTR)
  • Baud Rate: 9600
  • Data Bits: 8
  • Parity: None
  • Stop Bits: 1
  • Terminator: CR only (0x0D, NOT CR/LF!)
  • Interface Type: visa-serial
VisaIF Configuration Entry:
Arroyo-6301 ;; ArroyoInstruments;; ComboSource6301;; Other ;; visa-serial;; ASRL1::9600.8.1.none.none.CR.CR;; 1e4 ;; 1e4
